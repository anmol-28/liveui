# Live Dashboard

A simple React dashboard UI with static placeholders.

## Setup

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Structure

- `src/components/SummaryCard.jsx` - Summary card component
- `src/components/EventsTable.jsx` - Events table component
- `src/components/ChartPlaceholder.jsx` - Chart placeholder component
- `src/App.jsx` - Main app component
- `src/index.css` - Global styles
